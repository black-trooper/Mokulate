ファームウェアを書き込んだあと、まずは **有線（USB）での動作確認** を行い、その後に **電池を入れて無線での動作確認** をします。

---

# 1. 有線（USB）での動作確認

左右分割キーボードのため、左右それぞれを **個別に USB 接続して** 動作を確認します。  
左右をケーブルでつなぐ必要はありません。

1. **左側を USB 接続する**
2. オンラインキーテスターで左側のキー24個がすべて反応するか確認  
   - https://config.qmk.fm/#/test
3. **右側も同様に USB 接続し、同じテストを行う**

チェックポイント：
- 全キーが押したとおり反応する  
- 反応しないキーがない  
- 意図しない入力が出ない（チャタリング確認）  

---

# 2. 無線（バッテリー）での動作確認

有線で問題がなければ、次に無線の確認を行います。

1. 左右両方に電池をセット  
2. スライドスイッチを **ON（中央側）** にする  
3. PC / スマホの Bluetooth 設定にデバイス名が表示される  
4. ペアリング後、キーテスターで再度動作確認

---

# 3. 不具合があった場合のチェックリスト

- 正しい side（left / right）のファームを書き込んだか  
- .uf2 をもう一度書き込んでみる  
- 電池の極性は間違えていないか
- スイッチの向き（ON/OFF）を確認  
- USB ケーブルやポートの不良ではないか
