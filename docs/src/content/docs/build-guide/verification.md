ファームウェアを書き込んだあとは、電池を入れて無線での動作確認をします。

## 無線での動作確認

1. 左右両方に電池をセット  
2. スライドスイッチを **ON（中央側）** にする  
3. PC / スマホの Bluetooth 設定にデバイス名が表示される  
4. ペアリング後、オンラインキーテスターで48個のキーがすべて反応するか確認  
   - https://config.qmk.fm/#/test

チェックポイント：
- 全キーが押したとおり反応する  
- 意図しない入力が出ない（チャタリング確認）  

---

## 不具合があった場合のチェックリスト

- 正しい side（left / right）のファームウェアを書き込んでいるか  
- 電池の極性を間違えていないか  
- 電源スイッチが ON になっているか  
- 一部のキーが反応しない場合はソケットのはんだ付け不良の可能性  
- 特定の列・行がまとめて反応しない場合は、マイコン周辺のはんだ付け不良の可能性
