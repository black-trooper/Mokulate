# Mokulate

![Mokulate](docs/public/hero.jpg)

## Features

- 🪵 無垢材を贅沢に削り出したトップケース
- 🔋 電池駆動でケーブルレスな分割キーボード
- 🏕️ 手首に自然な角度を与えるテンティング機構
- ⌨️ すっきりと美しいオーソリニア配列
- ☁️ タイピング時にわずかにキーが沈む *tadpole* マウント
- 🥷 電池もマイコンも一切露出しないシンプルデザイン
